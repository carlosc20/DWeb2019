Abstract:

Este projeto consiste em várias partes, usando o dataset dos arqueossítios em XML:

Uma stylesheet XSLT que processe este ficheiro e gere um ficheiro por cada arqueossítio e os coloque numa pasta: arq1.xsml, arq2.xml...

Uma stylesheet que esteja associada aos ficheiros XML gerados e os transforme em HTML.

Um servidor em nodejs que serva os ficheiros, ou seja, responde aos pedidos: localhost:7777/3 --> Coloca no browser o conteúdo do arq3.xml em formato HTML.

Deliverables:

  • XSL usado para garar o XML dos arqueossítios individuais (XSL)
  • XSL usado para gerar HTML dos arqueossítios (XSL)
  • Servidor node.js (JS)
